🚀 AI Application – Algorithm Visualizer & Comparison Tool
📌 Overview
This project is a web-based AI application that demonstrates and compares different pathfinding algorithms like:

Breadth First Search (BFS)
Depth First Search (DFS)
A* Algorithm
It helps users visually understand how each algorithm works and compares their performance.
